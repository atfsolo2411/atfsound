# 🎵 Магазин автозвука

React-приложение для продажи автоакустики с корзиной и фильтрацией.

## 🚀 Функции
- Товары по категориям (динамики, сабвуферы, усилители)
- Добавление/удаление из корзины
- Подсчет суммы
- Адаптивный дизайн

## 🛠 Стек
React, JavaScript, CSS3

## ⚡ Запуск
```bash
npm install
npm start
