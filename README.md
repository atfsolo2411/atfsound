# 🎵 Магазин автозвука

Интернет-магазин автомобильной акустики на React. Корзина, фильтрация по категориям, адаптивный дизайн.

## ✨ Функционал

- 🛒 Добавление товаров в корзину
- 🗑 Удаление из корзины
- 📊 Фильтрация по категориям
- 💰 Подсчет итоговой суммы
- 🔔 Уведомления при добавлении
- 📱 Адаптивный дизайн

## 🛠 Технологии

- **React** (классовые компоненты)
- **JavaScript ES6+**
- **CSS3**
